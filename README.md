# Proyecto Frontend - Banco de Agricultura

Este es el repositorio del código frontend para el proyecto del Banco de Agricultura.

Este proyecto fue generado con **Angular CLI** versión **16.2.0**.

---

##  Prerrequisitos

Antes de poder ejecutar este proyecto, se necesita tener instalado:

### 1. Node.js y NPM

El proyecto requiere **Node.js** (que incluye **NPM**). Angular 16 es compatible con:

* **Node.js:** Versión `v16.x` o `v18.x` (Se recomienda v18 LTS).

NPM (Node Package Manager) se instalará automáticamente junto con Node.js.

### 2. Angular CLI (Opcional, pero recomendado)

Para facilitar la ejecución de comandos de Angular, puedes instalar el Angular CLI de forma global:

```bash
npm install -g @angular/cli
